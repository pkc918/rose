# 相册图片存放说明

## 文件夹结构

请按照以下格式组织你的图片：

```
public/photos/
├── 2025-10/          # 2025年10月的图片
│   ├── photo1.jpg
│   ├── photo2.png
│   └── ...
├── 2025-09/          # 2025年9月的图片
│   ├── photo1.jpg
│   └── ...
└── 2024-12/          # 2024年12月的图片
    └── ...
```

## 命名规则

1. **文件夹命名**：必须使用 `YYYY-MM` 格式（例如：`2025-10`）
2. **图片格式**：支持 `.jpg`、`.jpeg`、`.png`、`.gif`、`.webp` 格式
3. **图片命名**：可以使用任意文件名

## 排序规则

- 月份文件夹按时间倒序排列（最新的月份在最前面）
- 同一月份内的图片按文件名排序

## 示例

创建新月份的图片：

```bash
# 创建 2025年11月的文件夹
mkdir public/photos/2025-11

# 将图片复制到文件夹中
cp ~/Downloads/my-photo.jpg public/photos/2025-11/
```

访问相册页面：`http://localhost:3000/photos`

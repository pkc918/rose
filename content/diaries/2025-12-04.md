date: 2025-12-04

## Q?
怎么样同时能使用数组和对象解构返回值？

## A!
参考 [MDN 文档](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Symbol/iterator)

使用 Generator Function + Symbol.iterator 实现

```ts
function useToggle(value: boolean) {

  function toggle() {
    value = !value
    return value
  }

  function toggleTrue() {
    value = true
    return true
  }

  function toggleFalse() {
    value = false
    return false
  }

  return {
    toggle,
    toggleTrue,
    toggleFalse,

    *[Symbol.iterator](){
      yield toggle
      yield toggleTrue
      yield toggleFalse
    }
  }
}
```
